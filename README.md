# WEEK 1 – Bell Animation

## About
This project is an interactive bell animation created using HTML, CSS, and JavaScript as part of the Learners' Space assignment.

## Features
- Bell SVG animation
- Hover interaction
- Bell moves slightly away from the cursor
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- SVG

## How to Run
1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
